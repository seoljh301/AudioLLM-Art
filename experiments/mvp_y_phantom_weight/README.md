# MVP-Y — Phantom Weight

**위치:** matrix `∅ → audio · weight`.

## 가설
RAVE 가중치 자체에 부여된 "사전 분포"가 어떤 소리인지 듣는다. 입력 = 0, latent 분포 = random walk, weight = 두 체크포인트의 morph t. D 의 입력 없는 쌍대이자 O 의 고정 가중치 형제.

## 사용 예
```bash
bash experiments/mvp_y_phantom_weight/run.sh --out runs/mvp_y/phantom_t05_10s.wav
```
